Blueprint 是一个基于 PaperMod 的 Hugo 技术写作主题。这个示例站展示安装说明、Markdown 排版、代码块、搜索、标签、归档和语言切换。

## 示例范围

| 页面 | 用途 |
|---|---|
| 首页 | 最近文章 |
| 文章页 | 带紧凑元数据的阅读视图 |
| 归档 | 按时间回看内容 |
| 搜索 | 本地 JSON 搜索 |
| 标签 | 主题聚合 |

## 适合场景

Blueprint 适合研究笔记、工程日志、文档札记、课程笔记和个人知识库。
